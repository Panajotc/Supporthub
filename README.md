# SupportHub

SupportHub is a production-style ticketing and support management system built with Laravel, React, TypeScript, MySQL, and Laravel Sanctum.

The project is designed to demonstrate real-world support workflows, role-based access control, REST API design, clean backend architecture, and full-stack engineering practices.

## Tech Stack

### Backend

- Laravel 12
- PHP 8+
- MySQL
- Laravel Sanctum

### Frontend

- React
- TypeScript
- Tailwind CSS
- React Router
- Axios

## Core Features

- Authentication
- Role-based access control
- Ticket creation and management
- Ticket replies
- Assignment workflow
- Status and priority management
- Ticket status history

## Roles

- Customer
- Agent
- Admin
